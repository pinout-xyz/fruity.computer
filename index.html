I really have absolutely no idea what I'm doing with this yet!

Hello. I've been working with a certain fruit-themed computer more or less since it launched.

I cut my teeth (that's a silly expression!) packaging up WiringPi as a Ruby module using SWIG and various other incantations I didn't understand.

Many years later I've learned a lot - mostly about how much more I don't understand - and entrenched myself into the zeitgeist of embedded computing and open-source code.

But I have made a bunch of stuff I'd like to share!

Pinouts and things...

  • A minimal GPIO pinout for the Raspberry Pi (40-pin) SBC.
  • An accessible, interactive pinout for the Raspberry Pi Pico microcontroller board.
  • A Raspberry Pi Pico pinout desktop wallpaper.

FOSS projects...

  • ffsmake - a tool for packing up MicroPython-compatible FAT filesystems
  • dir2uf2 - a tool for packing up MicroPython-compatible LittleFs filesystems
  • py_decl - a Python library for reading Pico SDK's "binary declaration" metadata.
